Latin Square Experiment

Author

Axel Baudot

Latin Square : choosing the heuristic with max product BP

  • 1st solution: both find solutions close to optimal quite fast, max-marginal-regret sometime finds better, even optimal solutions first.
  • last solution: both improve solution fast, max-marginal-regret finds a few instance “difficult” with a disproportionate number of fails
  • optimality proof: max-marginal-regret proves optimality with 2-10x more fails, + a few “difficult” instances

Domwdeg seems more robust overall, especially if an optimality proof is required. Max-marginal-regret might be useful to find one good solution solution fast, and no optimality proof.
