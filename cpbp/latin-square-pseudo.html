Latin Square Experiment

Author

Axel Baudot

Latin Square Experiment : LDS, pseudodiagonal

Latin Square are turned into a COP as we try to maximize the sum of the “pseudodiagonal” : a rectangular area of 2*15 cells.

Different models and branching schemes are compared on the same instances.

  • sum_product-max_marginal_regret: bp with sum product, branching with max marginal regret
  • sum_product-max_marginal_regret-oracle: bp with sum product, oracle on objective, branching with max marginal regret
  • max_product-max_marginal_regret-oracle: bp with max product, oracle on objective, branching with max marginal regret
  • max_product-dom_wdeg_max_marginal-oracle: bp with max product, oracle on objective, variable selection with dom_wdeg, value selection with max_marginal
  • no_bp-max_value: select the max value on the diagonal
Code 
import plot

instance_numbers = range(1, 11)

models1 = [
  'lds-sum_product-max_marginal_regret',
  'lds-max_product-max_marginal_regret-oracle',
  'lds-max_product-dom_wdeg_max_marginal-oracle',
  'lds-sum_product-dom_wdeg_max_marginal',
  'lds-no_bp-max_value',
]
models2 = [
  'lds-max_product-max_marginal_regret-oracle',
  'lds-max_product-dom_wdeg_max_marginal-oracle',
  'lds-sum_product-dom_wdeg_max_marginal',
]
